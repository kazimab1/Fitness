# Fitness static package - tracker-style home

This package keeps the static HTML import architecture but changes the home screen to match the simpler Fitness_Tracker shell more closely.

## Files
- index.html - app entry point for GitHub Pages
- style.css - styling for the tracker shell and all tabs
- app.js - page rendering, nav, import logic, home progress controls
- parser.js - parses imported JSX/HTML/JSON plan files
- data/default-plan.json - fallback plan shown before import
